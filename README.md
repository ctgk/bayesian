# Bayesian Inference

Collection of Bayesian inference methods

# Running App

```
$ pip install git+https://github.com/ctgk/bayesian#egg=bayesian[app]
$ streamlit run https://raw.githubusercontent.com/ctgk/bayesian/master/app.py
```

# Contributing

$ cd bayesian
$ pre-commit install
